Compute — AWS Lambda (Node.js 22)
Lambda · Triggered by SES
email-processor
report-builder-email-processor-{env}

1. Parse raw email via mailparser
2. Look up property slug from SSM email-mapping
3. Store attachment to S3 incoming-files with UUID filename
4. Handles deduplication via unique filename
Lambda · Triggered by EventBridge
file-processor
report-builder-file-processor-{env}

1. Discover last-24h files in S3 incoming-files
2. Group by property slug
3. Check processed-markers for duplicates
4. Route to Visual Matrix or Opera pipeline
5. Load mapping XLSX from S3
6. Parse → transform → generate JE + StatJE
7. Write to S3 processed-files
8. Send reports via SES
